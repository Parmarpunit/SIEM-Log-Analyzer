# SIEM Log Analyzer

A log analysis tool for detecting suspicious activity in system/network logs, inspired by real-world SOC workflows.
Purpose
Built to practice log correlation and anomaly detection — the kind of analysis a SOC Analyst performs when triaging alerts.

Tech Stack
- Python Splunk 

Features
-Parses Windows Event Logs / Sysmon logs
-Flags brute-force login attempts
-Generates summary reports of suspicious events



What I Learned
Understanding how raw logs translate into actionable security alerts, and the importance of context (not just matching patterns) when investigating events.

Future Improvements

-Add MITRE ATT&CK technique mapping
-Real-time log ingestion
